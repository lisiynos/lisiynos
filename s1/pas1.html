Создан для обучения программированию, 1968 год, швейцарский ученый Никлаус Вирт, название язык получил в честь создателя первой механической вычислительной машины Блеза Паскаля.

1985 год - фирма Borland - Turbo Pascal версии 3.0, 1992 год - последняя версия языка Turbo Pascal - 7.0, Object Pascal – Delphi, Object Pascal для платформы .NET, Oxygene для платформы .NET, Modula-2 – Oberon – Java.

Delphi Prism для платформы .NET от компании Embarcadero.

Комментарии

Комментарии — пояснения к исходному тексту программы внутри текста программы. Синтаксис комментариев определяется языком программирования. Комментарии не оказывают никакого влияния на результат компиляции программы или её работу.

Однострочный комментарий - от текущего места до конца строки, многострочный комментарий - начинается с заданной комбинации символов и заканчивается другой заданной.

Синтаксис комментария в Pascal/Delphi, вложение комментариев:

// Однострочный комментарий Delphi
{ Многострочный комментарий Pascal }
(* Многострочный
    комментарий Pascal *)
(*  Начало длинного комментария Pascal
     { Вложенный комментарий Pascal }  
    { Ещё один вложенный комментарий Pascal }  
    Конец комментария Pascal
 *)

Использование комментариев:

  • пояснения к тексту программы (неочевидные из текста программы моменты).
  • временное отключение фрагментов текста программы (закомментировать на время кусок программы, например, для отладки).

Хорошая практика: писать комментарии до текста программы, чтобы точно сформулировать способы и особенности реализации перед самой реализацией.

Примеры:

begin
  a := 2;
  b := 3;
  if (a <> b) then
    inc(a);
  if (a = b) then
    c := a
  else if (a < b) then
    c := a + 1
  else
    c := a - 1;
end.

Циклические алгоритмы

Цикл на заданное число повторений (инструкция FOR)

Выполняет тело цикла указанное количество раз.

for i := 1 to 10 do writeln(i);

Выведет на экран 10 чисел от 1 до 10. переменная i перебирает все целые числа от первого аргумента до второго,оба включительно. i должна быть типа integer. Int64 нельзя использовать в for (как переменную цикла).

Цикл с предусловием (инструкция WHILE)

while (пока тут правда) он делает что-то

begin
  a := 1;
  while (a <= 10) do begin
    writeln(a);
    inc(a);
  end;
end.

Выведет 10 чисел

Оператор Break перывает цикл(любого вида(for,while)).Оператор Continue переходит к следующей итерации цикла.

Массивы

Массив — упорядоченный набор данных, для хранения данных одного типа, идентифицируемых с помощью одного или нескольких индексов. В простейшем случае массив имеет постоянную длину и хранит единицы данных одного и того же типа. Количество используемых индексов массива может быть различным. Массивы с одним индексом называют одномерными, с двумя — двумерными и т. д. Одномерный массив нестрого соответствует вектору в математике, двумерный — матрице. Чаще всего применяются массивы с одним или двумя индексами, реже — с тремя, ещё большее количество индексов встречается крайне редко.

Индекс массива — целое число, либо значение типа, приводимого к целому, указывающее на конкретный элемент массива.

Пример: 

var
  a:array[1..105] of integer;
begin
  a[1] := 2;
  a := 3 -> Compilition Error
  for i := 1 to 5 do
    a[i] := 5 - i + 1;
  s := 0;
  for i := 1 to 5 do
    s := s + a[i];
  writeln(s); // 15
  writeln(a[4]); // 2
  writeln(a[1] + a[5]); //6
end.
